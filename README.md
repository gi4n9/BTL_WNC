## Getting Started

Yêu cầu phần mềm:
MySql
Nodejs
Code Editor

1. Tạo file .env trong /backend
2. Import database trong MySQL
3. Tạo 2 terminal
4. Terminal 1: cd backend -> npm i -> npm run start.
5. Terminal 2: npm i -> npm run dev.

# backend/.env

PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_database_password
DB_NAME=your_database_name # nên đặt là barberbooking
JWT_SECRET=your_secret_jwt_key

Chúc thầy may mắn!
